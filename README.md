
## Как запустить на локальной машине


Clone the project

```bash
 git clone https://github.com/htnr76/guts.git
```

Go to the project directory

```bash
  cd guts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
